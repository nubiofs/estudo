Certificados para chamar Web Services da Receita

Primeiramente deve-se gerar um arquivo de certificado digital junto à SUNAC. Seguir o procedimento descrito neste link. Após, seguir com os passos abaixo, se necessário (alguns dos passos podem já constar do tutorial do link anteriormente citado).

A partir do certificado (arquivo .CER, neste exemplo chamado meu-certificado.cer) gerado junto à SUNAC, execute o seguinte comando:

 

openssl pkcs12 -export -out cadeia.pfx -in meu-certificado.cer -inkey /etc/ssl/chave.key

Observe que o comando vai solicitar a senha da chave privada. Em seguida, você fornecerá uma senha para exportar a cadeia (exemplo: xyz123).

 

(O arquivo /etc/ssl/chave.key foi gerado anteriormente para gerar o requisicao.csr, enviado à SUNAC no formulário disponível em https://ccdhom.serpro.gov.br/serproacf/ .)

O resultado deste comando será o arquivo cadeia.pfx. Para testar este arquivo, siga os seguintes passos:

  1. No Firefox, clique em Editar/Preferências/Aba Avançado/Aba Criptografia.
  2. Clique no botão Certificados/Aba Seus Certificados.
  3. Clique no botão Importar e importe o arquivo cadeia.pfx gerado. Agora ele vai pedir aquela senha que você forneceu no comando openssl pkcs12 (exemplo: xyz123).

 Isto instalou a cadeia de certificados no Firefox. Agora acesse os Web Services e deverá funcionar. Para ter certeza, remova o token (se estiver inserido).

https://hom-infoconv.receita.fazenda.gov.br/ws/cpf_atendimento/cpf_atendimento.asmx?wsdl

https://hom-infoconv.receita.fazenda.gov.br/ws/preatendimento/preatendimento.asmx?wsdl

Adicionar os certificados ao cacerts

Por padrão o Java tem um arquivo de certificados, que é o JAVA_HOME/jre/lib/security/cacerts. O arquivo da cadeia.pfx gerado anteriormente deve ser adicionado ao cacerts através dos seguintes comandos. Quando pedir a senha do cacerts, ela é por default igual a "changeit". Além disso, faça o download dos 3 certificados que estão na parte inferior da tela em https://ccdhom.serpro.gov.br/serproacf/opcoes/cadeia.html .

cd JAVA_HOME/jre/lib/security
keytool -importkeystore -destkeystore cacerts -srckeystore ~/Desktop/cadeia.pfx -srcstoretype PKCS12
keytool -import -alias raiz -keystore cacerts -trustcacerts -file /home/00838151043/Desktop/keys/icpbrasilv2.cer
keytool -import -alias acserpro -keystore cacerts -trustcacerts -file /home/00838151043/Desktop/keys/acserprov3.cer
keytool -import -alias serpro -keystore cacerts -trustcacerts -file /home/00838151043/Desktop/keys/serproacfv3.cer
