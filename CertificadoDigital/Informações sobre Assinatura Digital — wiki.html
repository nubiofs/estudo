Informações sobre Assinatura Digital

Página com compilação e estudo de informações relacionadas a assinatura digital: links para guias, orientações, tutoriais, ferramentas, definições, etc. Principalmente relacionado a programação Java.

http://docs.oracle.com/javase/1.5.0/docs/guide/security/cert3.html

Definições de termos sobre o assunto.

http://docs.oracle.com/javase/tutorial/security/apisign/index.html

Guia: Gerando e verificando assinaturas usando API Java.

http://docs.oracle.com/javase/1.3/docs/tooldocs/win32/keytool.html

Manual de uso da ferramenta keytool, com definições sobre o que é KeyStore e relacionados.

http://www.devdaily.com/java/java-using-keytool-certificate-file-create

Explicação simples, exemplificada, de como funciona uma assinatura e verificação digital, usando keytool. Os links para artigos relacionados do texto possuem mais exemplos sobre keytool.

http://conshell.net/wiki/index.php/OpenSSL_to_Keytool_Conversion_tips
http://www.agentbob.info/agentbob/79-AB.html

Convertendo chave e certificado do formato openssl (PEM ou DER) para o formato keystore (JKS).

http://conshell.net/wiki/index.php/Keytool_to_OpenSSL_Conversion_tips
http://coreygilmore.com/blog/2009/06/03/export-a-private-key-from-a-java-keystore/
http://www.zimbra.com/forums/administrators/9832-exporting-private-key-keystore-use-postfix-apache.html

Obtendo certificado e chave primária vindo de uma keystore JKS.

http://www.vanemery.com/Linux/Apache/apache-SSL.html

Tutorial como criar um par chave pública/privata e um certificado auto-assinado para um servidor Apache usando a o comando openssl.
