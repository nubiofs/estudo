        Hi Paul. Nice and useful blog post - as always. One hickup IMHO remains in order to use your handy function e.g. in a command line with psql like this psql -U user -d mydb -f query.sql > out.geojson.. Currently it spits out:


        { "type": "Feature", ... "properties": {...}}
        { "type": "Feature", ... "properties": {...}}
        { "type": "Feature", ... "properties": {...}}
        ...

        But what GeoJSON wants, is this (see FeatureCollection wrapper object and array commas):


        { "type": "FeatureCollection",
        "features": [
        { "type": "Feature", ... "properties": {...}},
        { "type": "Feature", ... "properties": {...}},
        { "type": "Feature", ... "properties": {...}}
        ...
        ]
        }

        So, perhaps a final jsonb_build_object('FeatureCollection', ...) is needed?
