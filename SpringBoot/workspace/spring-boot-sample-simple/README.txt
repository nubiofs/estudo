* Configurações (inicialização do projeto):

$ cd estudo.git/SpringBoot/workspace/spring-boot-sample-simple

$ mvn eclipse:eclipse

$ mvn clean install

$ java -jar target/spring-boot-sample-simple-1.5.4.RELEASE.jar

* Criando (Maven Wrapper) para o projeto:

$ mvn -N io.takari:maven:wrapper

$ ./mvnw clean install

$ ./mvnw spring-boot:run
